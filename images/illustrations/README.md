- `case-detail-illustration.svg`
  - Source idea: `CaseDetails.png`

- `turnover-analytics-illustration.svg`
  - Source idea: `Turnover-Analysis.png`

## Notes

- Use `privacy-illustration.png` instead of the earlier generated privacy SVG.
- The SVG assets are editable and can be tuned directly in code.
- The privacy PNG is a finalized illustration asset rather than an editable vector.
- Labels and numbers in the SVGs are illustrative, not product screenshots.

## Optional Sora / Nano Banana Follow-Up

If you want more atmospheric or motion-heavy variants later, use these SVGs as the design reference rather than the raw UI screenshots. That will give the video/image generator a cleaner visual target and reduce dashboard clutter.
